# MCP Demo

> Coming soon — this folder is a placeholder for an env0 MCP (Model Context Protocol) integration demo.

## Planned Demo

This demo will show how to use env0 with an MCP server to enable AI agents (Claude, Cursor, etc.) to trigger infrastructure deployments, query environment status, and manage env0 resources through natural language.

## What MCP Enables

With an env0 MCP server, an AI assistant can:
- List environments and their deployment status
- Trigger deployments or destroy operations
- Fetch deployment logs and outputs
- Create and update template variables
